
# AGENCY OS - SYSTEM ROOT

This directory contains the autonomous AI agency system.

## STRUCTURE
- `_SYSTEM/` : The Engine, Personas, and Dashboard. DO NOT TOUCH.
- `PROJECTS/` : All output work (Client Projects) lives here.
- `00_BRIEF/` : Drop new project briefs here to trigger the system.

## HOW TO USE
1. Add a brief to `00_BRIEF/`.
2. The System wakes up.
3. Watch `PROJECTS/[YourProjectName]` fill with files.
